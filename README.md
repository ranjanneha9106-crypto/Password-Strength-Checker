# Cybersecurity Learning

This repository contains beginner cybersecurity projects created while learning cybersecurity fundamentals and Python programming.

## Projects

### Password Strength Checker
This tool checks whether a password is:
- Weak
- Medium
- Strong

The analysis is based on:
- Password length
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

### Basic Port Scanner
This tool scans a target system and identifies open ports.

- Python Programming
- Cybersecurity Fundamentals
- Network Security Basics
